1. khai báo package.json ==> npm init -y
2. cài đặt webpack ==> npm i -d webpack webpack-cli webpack-dev-server
3. định nghĩa câu npm start trong file package.json ==>"Start": "webpack serve --open"
4. cài đặt file json   npm install -g json-server
5. sau khi cài json, json-server --watch db.json hoặc npm run start:server để biến file data.json thành link giống mockapi
